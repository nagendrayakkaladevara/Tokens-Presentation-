What is a Token?

A token is the smallest meaningful text unit an LLM processes.

The model does NOT read full sentences directly.

Sentence
→
Broken into Tokens
→
Processed by AI

Example

Sentence: I love Angular

Possible tokens:

I love Angular

Notice:

Tokens may contain spaces

This is VERY important.

Tokens Are NOT Always Words

Example

unbelievable

may become:

un believ able

Tokens are not always letters too.

Code Tokenization Example

TypeScript 
const user = fetchData();

Possible tokens:

Token
  • const
  • user
  • =
  • fetch
  • Data
  • ()
  • ;
